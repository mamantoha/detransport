# Detransport

## Installation

* Install Ruby
* Install Node.js
* Install Yarn

### Clone from GiHub

```console
git clone https://github.com/mamantoha/detransport.git
cd detransport
```

### Install Ruby dependencies

```console
bundle install
```

### Install npm dependencies

```console
yarn
```

### Run development server

```console
bundle exec rackup
```

```console
npm run build
```
